# Tic-Tac-Toe Web Application

## Project Overview

This project is a Tic-Tac-Toe Web Application developed as part of the SkillCraft Technology Web Development Internship. The application allows users to play the classic Tic-Tac-Toe game either against another player or against the computer.

The project focuses on implementing game logic, user interaction, state management, and winning condition detection while providing an engaging and responsive user experience.

## Features

* Interactive 3x3 Tic-Tac-Toe game board
* Player vs Player mode
* Player vs Computer mode
* Real-time turn tracking
* Automatic win and draw detection
* Winning combination highlighting
* Restart game functionality
* Responsive design for desktop and mobile devices
* Smooth animations and hover effects
* User-friendly interface

## Technologies Used

* HTML5

* JavaScript

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using a web browser or Live Server extension.
4. Select a game mode and start playing.

## Game Rules

1. The game is played on a 3×3 grid.
2. Players take turns marking cells with X and O.
3. The first player to align three marks horizontally, vertically, or diagonally wins.
4. If all cells are filled without a winner, the game ends in a draw.
